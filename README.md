## 📌 Project Overview  
A decentralized organ donation system that ensures transparency, security, and fairness using **blockchain technology**.

✅ Enable **donors** to pledge organs securely.  
✅ Allow **recipients** to register and get matched based on medical criteria.  
✅ Ensure **tamper-proof records** using smart contracts.  
✅ Provide real-time notifications and updates.
